# Exam Answer Template Generator App

This project now includes both:
- a **web app** (Flask) for interactive use in the browser
- a **CLI** for terminal-based usage

## Run the web app

```bash
python3 -m pip install flask
python3 app.py
```

Then open `http://127.0.0.1:5000`.

## Run the CLI

```bash
python3 exam_template.py "Explain the causes of the French Revolution."
```
